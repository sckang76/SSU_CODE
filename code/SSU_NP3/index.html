전기분야 (실기)

가닥수 '6가닥'의 법칙: 벨/표/응/발/전/공(6개). 여기에 7층/2000㎡ 초과시 우선경보방식 적용으로 '경종/표시등 공통선'이 추가되는 로직만 파악하면 가닥수 문제는 100% 맞힙니다. 도면 그릴 때 '종단저항' 표시 누락은 0점 처리되니 주의!

기계분야 (실기)

단위 환산의 함정: 펌프 동력 계산 시 토출량(Q)은 반드시 m³/sec 또는 m³/min인지 확인하세요. 시험 문제는 주로 ℓ/min으로 주어집니다. 1,000으로 나누는 걸 잊는 순간 10점이 날아갑니다. Hazen-Williams 공식은 조도(C)값 120이 기본입니다.

위험물산업기사

특수인화물 순서: '에-이-아-이' (에테르, 이황화탄소, 아세트알데히드, 이소프렌). 인화점이 낮은 순서대로 입에 붙여두세요. 반응식은 '화학식'을 먼저 적고 좌우 원자 수를 맞추는 연습만 5번 하면 충분합니다.

시험 종료 10분 전 체크리스트

  • 단위 기재 여부 확인 (kg, m, MPa 등 안 적으면 오답 처리)
  • 산출 근거에 소수점 처리 기준(둘째 자리 등) 준수 확인
  • 실기 도면 연필선 완벽하게 지웠는지 확인
