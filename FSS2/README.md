# 소방용어 검색 포털

엑셀로 정리한 소방용어를 검색할 수 있는 Vite + React 앱입니다.

## 실행 방법

1. 의존성 설치

```bash
npm install
```

2. 개발 서버 실행

```bash
npm run dev
```

3. 브라우저에서 Vite 주소 열기

## 사용 방법

- 상단에서 엑셀 파일을 업로드합니다.
- `파일 선택` 버튼으로 `.xlsx`, `.xls`, `.csv` 파일을 읽을 수 있습니다.
- 검색어를 입력하면 용어명, 정의, 분류, 약어, 영문명, 관련기준 기준으로 필터됩니다.
- `전체`, `용어명`, `정의`, `분류`, `약어`, `영문명`, `관련기준` 버튼으로 검색 범위를 바꿀 수 있습니다.
- `전체` 카테고리 또는 개별 분류 버튼으로 결과를 좁힐 수 있습니다.
- 결과 카드를 클릭하면 상세 정보가 바뀝니다.

## 엑셀 권장 컬럼

아래 이름 중 하나를 사용하면 자동으로 인식합니다.

- `용어명` 또는 `term`
- `정의` 또는 `설명`
- `분류` 또는 `category`
- `약어` 또는 `abbreviation`
- `영문명` 또는 `english`
- `관련법령`, `관련 기준`, `법령`, `reference`
- `태그`
- `상태`
